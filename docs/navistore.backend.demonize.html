Function demonize Try to demonize the current process.
def demonize(pidfile, server_start_func, *args):
Try to demonize the current process. If there was a failure writing the pid file, exit and stop execution to prevent demonizing without knowing the pid of the daemon.

If extra argument need to be passed to the server_start_func, pass them through the *args variable length argument array.
