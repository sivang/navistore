HTTP storage backend client test case.

This tests the HttpStorageBackend class, according to the interface conformance we defined in StorageBackendTestCase.

This test also makes the http server we work against, conforms to the navistore backend interface specification and actually serves to test the http server as well.

Since replication support is outside of the defined interface, and is part of the http server implementation we use a different test to test it, testServerReplication.py
